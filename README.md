# CanvasClaw

CanvasClaw is the standalone OpenClaw canvas UI extracted from local workspace runtime and prepared for iterative upgrades.

## Goal
A beautified, user-friendly canvas interface for users who want live dashboard/data-stream visualization without deep setup complexity.

## Contents
- `index.html` — main canvas UI
- `activity.js` — activity feed logic
- `cronCurrent.js` — current cron/task visualization logic

## Notes
This repository is intended to evolve as a reusable canvas product surface.
